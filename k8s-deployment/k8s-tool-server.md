### 🛠️ 모든 노드에 설치해야 하는 3대 컴포넌트

| 컴포넌트명 | 역할 설명 |
| --- | --- |
| **`kubeadm`** | 쿠버네티스 클러스터를 쉽고 빠르게 부트스트랩(구축 및 설정) 핵심 도구입니다. |
| **`kubelet`** | 클러스터의 각 노드에서 실행되는 에이전트로, 파드(Pod) 내 컨테이너들이 정상적으로 실행되고 관리되도록 제어합니다. |
| **`kubectl`** | 쿠버네티스 클러스터에 명령을 내리고 제어하기 위한 CLI(커맨드 라인 인터페이스) 도구입니다. |


### 상세 정보 조회를 통한 구조 분석 (`-o wide`)

`-o wide` 옵션을 추가하면 파드가 어느 서버(노드)에 배포되었는지, 어떤 IP를 할당받았는지 등 상세한 네트워크 및 컨테이너 정보를 한눈에 확인할 수 있습니다.

* **실행 명령어:**
```bash
kubectl get deploy,rs,po -o wide

```
